Open Data Hub Knowledge Graph

Access the rich high quality data from the Open Data Hub by a flexible standards compliant query language. SPARQL is the query language developed by the W3C to access huge amounts of data over the web and is used by many of the largest open data providers worldwide, as Wikidata, DBpedia, European Environment Agency and many more. The Open Data Hub is a platform to use data and create innovative solutions for the SMART Green Region South Tyrol.

Why choose this service

The use of a query language gives an extremely powerful and flexible tool to explore the data available from the OpenDataHub without having to download large amounts of data.

Data can already be combined and filtered on the server, so all the heavy duty work is kept off your computer and only the truly wanted data is transferred..

Being a W3C standard, support for SPARQL is provided by almost any tool and programming language used for working with data.
